## Overview

Genesis evolves rapidly to deliver better visual learning experiences. You receive automatic updates in your dashboard, with full details here. Review changes by version to understand new capabilities for creating customized lessons, stories, and ebooks.

## Recent Updates

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added sci-fi and fantasy visual style presets for immersive story worlds
- Introduced collaborative editing: Share lessons with team members via secure links
- New ebook export formats including EPUB with embedded visuals

## Improvements

- Enhanced AI prompt refinement: Suggests improvements to your story ideas automatically
- Faster generation times by `>30%` for children's stories and brand narratives

## Bug Fixes

- Fixed visual style inconsistencies in multi-page lessons
- Resolved export errors for high-resolution images

</Update>

## Stay Updated

Follow these steps to ensure you always have the latest Genesis features.

<Steps>
  <Step title="Enable Notifications" icon="bell">
    In your dashboard, toggle notifications under Account > Preferences.
  </Step>
  <Step title="Check Release Notes" icon="book-open">
    Visit this changelog after login to review changes.
  </Step>
  <Step title="Update Integrations" icon="settings">
    <Tabs>
      <Tab title="API" icon="code">
        Update your client code:

````javascript
// New endpoint
const response = await fetch('https://api.genesis.app/v1/generate', {
  method: 'POST',
  headers: { 'Authorization': `Bearer ${token}` },
  body: JSON.stringify({ prompt: 'Your story idea', style: 'sci-fi' })
});
````

      </Tab>
      <Tab title="SDK" icon="package">
        Run the update command:

````bash
npm update @genesis/sdk
````

      </Tab>
    </Tabs>
  </Step>
</Steps>

<ExpandableGroup>
  <Expandable title="Detailed Migration Guide for v1.0 to v1.2" default-open="false">

    If you used legacy endpoints, update as follows:

    | Old Endpoint       | New Endpoint          | Changes                  |
    |--------------------|-----------------------|--------------------------|
    | `/legacy/generate` | `/v1/generate`       | Added `style` parameter |
    | `/export/pdf`      | `/v1/export/epub`    | Supports multiple formats |

    Test in the sandbox environment first.

  </Expandable>
</ExpandableGroup>
